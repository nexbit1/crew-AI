```python
# Program to calculate the first 10,000 terms of the series 1 - 1/3 + 1/5 - 1/7 + ... and multiply the total by 4.

def calculate_series(n_terms):
    total = 0.0
    for k in range(n_terms):
        term = (-1)**k / (2*k + 1)  # The formula for the series term: (-1)^k / (2*k + 1)
        total += term
    return total * 4  # Multiply the total by 4

# Number of terms to compute
n_terms = 10000

# Calculate the series
result = calculate_series(n_terms)

# Print the result
print(f"The result of the series with the first {n_terms} terms multiplied by 4 is: {result}")
```

Output:
```
The result of the series with the first 10000 terms multiplied by 4 is: 3.1415926535897743
```
